# Address Book Application in C

## Description
This project implements an Address Book system to manage contact details such as name, phone number, and email. It supports adding, searching, editing, deleting, and listing contacts using file handling.

## Features
- Add new contact
- Search contact
- Edit contact
- Delete contact
- Display all contacts
- File-based contact storage

## Technologies Used
- C Programming
- Structures
- File Handling
- Pointers
- Modular Programming

## Files
- main.c
- contact.c
- file.c
- populate.c
- contact.h
- file.h
- populate.h

## Compile
gcc main.c contact.c file.c populate.c -o addressbook

## Run
./addressbook
